# 🎯 Quiz Bot — battle.aiplus.kz

Автоматический бот для прохождения тестов по **Истории Казахстана** на сайте [battle.aiplus.kz](https://battle.aiplus.kz).

## 📋 Описание

Бот использует стратегию **«Учись и побеждай»**:
1. **1-й прогон** — выбирает случайный ответ и запоминает правильный (по зелёной подсветке)
2. **2-й+ прогон** — использует накопленный кэш ответов и набирает **90%+**

Бот проходит темы **4–10** и **12–39** (по 6 подтем в каждой), выполняя **5 успешных попыток** на каждую тему.

- **Python 3.10+** — [скачать с python.org](https://www.python.org/downloads/)
- **pip** — менеджер пакетов (устанавливается вместе с Python)

## 🚀 Установка и запуск

### 1. Установить Python

Скачай и установи Python с [python.org](https://www.python.org/downloads/).

> ⚠️ При установке **обязательно поставь галочку** `Add Python to PATH`

### 2. Открыть терминал

Открой **PowerShell** или **Командную строку (cmd)** и перейди в папку проекта:

```bash
cd C:\Users\kuany\Desktop\Сам
```

### 3. Установить зависимости

```bash
pip install playwright
```

### 4. Установить браузер для Playwright

```bash
python -m playwright install chromium
```

### 5. Запустить бота

```bash
python quiz_bot.py
```

## 📖 Как это работает

1. Бот открывает браузер Chromium и логинится на сайт
2. Переходит в раздел «История Казахстана»
3. Для каждой темы:
   - Открывает тему и начинает тест (20 вопросов)
   - Если ответ есть в кэше — выбирает его с задержкой 10 сек
   - Если ответа нет — выбирает случайный (быстро, 0.5 сек)
   - После проверки запоминает правильный ответ
   - Повторяет, пока не наберёт 90%+ пять раз

## ❓ Возможные проблемы

| Проблема | Решение |
|---|---|
| `pip` не найден | Переустанови Python с галочкой `Add to PATH` |
| Ошибка Playwright | Выполни `python -m playwright install chromium` |
| Бот не логинится | Проверь логин/пароль в `quiz_bot.py` |
| Тест не начинается | Сайт мог обновить вёрстку — проверь селекторы |
